클라우드 DB 초기 설정
1Supabase 프로젝트 만들기

supabase.com 접속 → 무료 회원가입 → New Project 클릭 → 프로젝트 이름 입력 (예: strawberry-farm) → 비밀번호 설정 후 생성

💡 무료 플랜으로 충분합니다 (DB 500MB, 월 2GB 전송)
2데이터베이스 테이블 생성

프로젝트 생성 후 왼쪽 메뉴 SQL Editor → New query → 아래 코드 전체 복사 후 Run

create table journal ( id bigint primary key, date text, weather text, tam text, tpm text, water text, wamt text, growth text, bee text, notes text, created_at timestamptz default now() ); create table harvest ( id bigint primary key, date text, time text, sp numeric default 0, go numeric default 0, no numeric default 0, chs jsonb default '[]'::jsonb, rev numeric default 0, created_at timestamptz default now() ); create table pest ( id bigint primary key, found text, treat text, type text, name text, chem text, amt text, safe text, notes text, created_at timestamptz default now() ); create table schedule ( id bigint primary key, date text, type text, title text, descr text, created_at timestamptz default now() ); alter table journal disable row level security; alter table harvest disable row level security; alter table pest disable row level security; alter table schedule disable row level security;
3연결 정보 입력

왼쪽 메뉴 Settings → API 에서 아래 정보를 복사하세요.
